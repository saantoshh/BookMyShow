🎬 BookMyShow Clone – Full Stack Movie Ticket Booking System

BookMyShow Clone – A full-stack movie ticket booking system built with Spring Boot (REST API) and Vanilla JS frontend. 
Features include user registration/login, movie browsing, show scheduling, seat selection, booking management, and cancellation. 
Backend uses Spring Data JPA with MySQL, structured with clean Controller-Service-Repository layers.

✨ Key Highlights
🔐 Secure Authentication – User registration & login with Spring Security

🎥 Movie Exploration – Browse movies with detailed information

🕒 Smart Show Scheduling – View shows across multiple screens & timings

🎟️ Interactive Seat Selection – Real-time seat booking experience

📦 Booking Management – Book, view, and cancel tickets seamlessly

👨‍💼 Admin Controls – Manage movies, shows, and theatre data

🧠 Architecture & Design

Designed with a clean and scalable architecture, following:

Controller → Service → Repository pattern
RESTful API design principles
Proper separation of concerns for maintainability

🛠️ Tech Stack

Backend: Java, Spring Boot, Spring Security, Spring Data JPA

Frontend: HTML, CSS, Vanilla JavaScript

Database: MySQL

Tools: Maven, Git, Postman

🎯 Objective

Built to showcase expertise in Java backend development, REST API design, and end-to-end application development.
